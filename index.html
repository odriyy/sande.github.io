Sand-E is one of the project initiated by SUTD Organization of Autonomous Robots aims to create a machine that can clean the beach. The Sand-E robot is powered by lithium-polymer (LiPo) batteries that is a fully remote-controlledcprototype.

Motivation Behind Sand-E

Focus on Sustainability

The initiation of this project was to complement SUTD’s inherent culture and emphasis on sustainability. SAND-E is made mostly of recycled or 3D printed parts, excluding the electronics.

Reducing Physical Strength

This project was taken up to reduce the physical strain experienced by our cleaners everyday, by automating the process.

Desire to Always be Innovative

This project is an evolution of the first prototype, driven by a strong desire to innovate

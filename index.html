Beach Cleaning Robot Project by Freshmore From SUTD

This project combinines engineering creativity to create a machine to tackle environmental problems. Together, we're making a difference for a sustainable future.

Introducing Sand-E !

Sand-E is one of the project initiated by SUTD Organization of Autonomous Robots (SOAR) aims to create a machine that can clean the beach. The Sand-E robot is powered by lithium-polymer (LiPo) batteries that is a fully remote-controlledcprototype.
